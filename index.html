О программе

ChatList — это удобное приложение для сравнения ответов различных AI-моделей. Отправляйте один и тот же промпт в несколько нейросетей одновременно и сравнивайте их ответы в удобной таблице.

Приложение поддерживает работу с OpenAI, DeepSeek, Groq, OpenRouter и другими провайдерами AI-моделей.

Основные возможности

🚀 Множественные запросы

  • Отправка промпта в несколько моделей одновременно
  • Сравнение ответов в реальном времени
  • Сохранение лучших результатов

💾 Управление данными

  • Сохранение промптов и результатов
  • Управление моделями и настройками
  • Поиск и сортировка по таблицам

🤖 AI-ассистент

  • Улучшение промптов с помощью AI
  • Генерация альтернативных вариантов
  • Адаптация для разных задач

🎨 Настройка интерфейса

  • Светлая и тёмная тема
  • Настройка размера шрифта
  • Удобный просмотр Markdown

📊 Экспорт данных

  • Экспорт результатов в Markdown
  • Экспорт в JSON
  • Копирование в буфер обмена

🔒 Безопасность

  • Хранение API-ключей в .env файле
  • Локальное хранение данных
  • Поддержка VPN и прокси

Установка

  1. Скачайте установщик с GitHub Releases
  2. Запустите файл ChatList-Setup-v1.0.0.exe
  3. Следуйте инструкциям установщика
  4. Настройте API-ключи в файле .env (см. документацию)

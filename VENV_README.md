# Виртуальное окружение ChatList

## Быстрый старт

### Активация виртуального окружения

**PowerShell:**
```powershell
.\venv\Scripts\Activate.ps1
```

Или используйте скрипт:
```powershell
.\activate.ps1
```

**Командная строка (CMD):**
```cmd
venv\Scripts\activate.bat
```

### Установка зависимостей

Зависимости уже установлены. Если нужно переустановить:
```powershell
pip install -r requirements.txt
```

### Запуск приложения

После активации виртуального окружения:
```powershell
python main.py
```

### Деактивация

```powershell
deactivate
```

## Установленные пакеты

- **PyQt5** (5.15.10) - GUI библиотека
- **pyinstaller** (6.3.0) - создание исполняемых файлов
- **requests** (2.32.5) - HTTP запросы
- **python-dotenv** (1.2.1) - управление переменными окружения
- **markdown** (3.10) - обработка Markdown
- **Pillow** (12.1.0) - работа с изображениями

## Примечания

- Виртуальное окружение использует Python 3.12.2
- Папка `venv/` исключена из системы контроля версий (см. `.gitignore`)
- При работе через VPN все зависимости установлены успешно

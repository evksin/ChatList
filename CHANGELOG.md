# Changelog

Все значимые изменения в проекте ChatList будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2025-01-XX

### Добавлено
- Первый релиз ChatList
- Отправка промптов в несколько AI-моделей одновременно
- Сравнение ответов в удобной таблице
- Сохранение промптов и результатов в базу данных SQLite
- Управление моделями (CRUD операции)
- Поддержка OpenAI, DeepSeek, Groq, OpenRouter
- AI-ассистент для улучшения промптов
- Генерация альтернативных вариантов промптов
- Адаптация промптов для разных типов задач
- Светлая и тёмная тема оформления
- Настройка размера шрифта
- Просмотр ответов в формате Markdown
- Экспорт результатов в Markdown и JSON
- Копирование в буфер обмена
- Поиск и сортировка по таблицам
- Логирование в файлы
- Инсталлятор с функцией удаления
- Поддержка VPN и прокси
- Обработка ошибок API с понятными сообщениями

### Исправлено
- Ошибка доступа к папке логов при установке в Program Files
- Проблемы с отображением версии в интерфейсе
- Ошибки при работе с некоторыми API провайдерами
- Проблемы с кодировкой в PowerShell скриптах

### Изменено
- Логи и база данных теперь хранятся в `%LocalAppData%\ChatList`
- Улучшена обработка ошибок сети
- Оптимизирована работа с базой данных

---

## Типы изменений

- **Добавлено** - для новых функций
- **Изменено** - для изменений в существующей функциональности
- **Устарело** - для функций, которые скоро будут удалены
- **Удалено** - для удалённых функций
- **Исправлено** - для исправления багов
- **Безопасность** - в случае уязвимостей

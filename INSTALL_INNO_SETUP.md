# Установка Inno Setup для создания инсталлятора

## Проблема

Для создания инсталлятора требуется программа **Inno Setup**, которая не установлена на вашем компьютере.

## Решение

### Вариант 1: Установить Inno Setup (рекомендуется)

1. **Скачайте Inno Setup:**
   - Перейдите на официальный сайт: https://jrsoftware.org/isinfo.php
   - Скачайте последнюю версию (рекомендуется версия 6.x или новее)

2. **Установите Inno Setup:**
   - Запустите скачанный установщик
   - Следуйте инструкциям установщика
   - Установите в стандартную папку (обычно `C:\Program Files (x86)\Inno Setup 6\`)

3. **Создайте инсталлятор:**
   ```powershell
   .\build_installer.ps1
   ```
   
   Или полная сборка:
   ```powershell
   .\build_all.ps1
   ```

### Вариант 2: Использовать только исполняемый файл

Если вам не нужен инсталлятор, вы можете использовать готовый исполняемый файл:

- **Расположение:** `dist\ChatList-v1.0.0.exe`
- **Использование:** Просто запустите файл двойным кликом

**Преимущества:**
- Не требует установки
- Можно запускать с USB-накопителя
- Не оставляет следов в системе

**Недостатки:**
- Нет ярлыков в меню Пуск
- Нет записи в "Установленные программы"
- Нет автоматического обновления

## Текущий статус

✅ **Исполняемый файл собран успешно:** `dist\ChatList-v1.0.0.exe`  
❌ **Инсталлятор не создан:** Требуется установка Inno Setup

## Проверка установки Inno Setup

После установки Inno Setup проверьте, что он найден:

```powershell
Test-Path "${env:ProgramFiles(x86)}\Inno Setup 6\ISCC.exe"
```

Или:

```powershell
Test-Path "${env:ProgramFiles}\Inno Setup 6\ISCC.exe"
```

Если команда вернёт `True`, Inno Setup установлен правильно.
